# Applied-GenAI_LLM-Apps_Design-Assistant-using-LangGraph
An AI-powered design assistant that transforms simple user inputs into high-quality cinematic visuals. It combines prompt engineering, workflow orchestration, and image generation to automate creative content production through an intuitive interface.

# Project Summary
In this project, we designed and developed an end-to-end AI-driven system that generates visually compelling campaign images from minimal user input. We structured the workflow using LangGraph to manage state and execution across multiple steps. User inputs (idea, genre, and style) are first enhanced into rich cinematic prompts using ChatOpenAI, ensuring detailed and context-aware descriptions. These prompts are then passed to gpt-image-1 to generate high-quality images, which are processed into displayable formats. The entire pipeline is exposed through an interactive UI built with Gradio, allowing users to easily generate and visualize outputs. By reducing user inputs to just three meaningful parameters while leveraging AI for prompt enrichment, the solution improves usability, consistency and scalability of visual content creation, effectively automating a traditionally manual and time-intensive design process.
